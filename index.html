Simple.Timer

A timer plugin for jQuery

Download .zip Download .tar.gz View on GitHub

Usage

Basic example: 

<script src="jquery.js"></script>
<script src="simple.timer.js"></script>
<script>
  $function(){
    $('.timer').startTimer();
  });
</script>

<div class="timer" data-seconds-left=800></div>

Use the onComplete option for custom behavior.

$('.timer').startTimer({
  onComplete: function(){
    console.log('Done!');
  }
});

Tests
