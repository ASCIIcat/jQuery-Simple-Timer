Usage

Basic example:

<div class="timer" data-seconds-left=800></div>

<script src="jquery.js"></script>
<script src="simple.timer.js"></script>
<script>
  $(function(){
    $('.timer').startTimer();
  });
</script>

Use the onComplete option for custom behavior:

$('.timer').startTimer({
  onComplete: function(){
    console.log('Done!');
  }
});
